# Graph Fraud Detection Frontend

This frontend is an **interactive experiment dashboard** for visualizing spectral effects in graph fraud detection under heterophily.

## Features
- One-click experiment execution
- Visualization of accuracy vs spectral cutoff
- Comparison of low-pass and high-pass filtering
- Baseline accuracy reference
- Clean, research-style UI

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Recharts

## How to Run

Install Dependencies:

```bash
npm install
```
Run the Code:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
